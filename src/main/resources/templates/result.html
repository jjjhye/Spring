다른 페이지에서 전달한 변수를 다시 사용가능

새로 저장한 변수도 사용가능(sum)

사용방번은 DTO명.변수명으로 지정해서 사용

DTO가 정상적으로 전달이 되면 DTO. 이후에 변수명이 나열되어야 한다.

일반태그에 출력

num1

num2

sum

직접 출력

num1 : [[${data.num1}]]>

num2 : [[${data.num2}]]>

sum : [[${data.sum}]]>

Form 태그에 출력

Form 태그에 object를 이용할 때 (검증처리)

object에 DTO이름을 지정, field에 변수명을 지정
